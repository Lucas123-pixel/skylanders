Trap Team Superchargers Imaginators


Skylanders:Imaginators foi o sexto e o ultimo jogo principal da série Skylanders, este jogo como o nome sugere é muito focado na imaginação do jogador dando a oportunidade dos jogadores criem seus próprios personagens.

Conhecidos como Imaginators que podem ser criados com uns cristais dentro de capsulas chamados creation crystals.

Também temos alguns Skylanders mais tradicionais com os Senseis, cada um deles é o mestre de uma das 10 classes de batalha do jogo e eles também ajudam a treinar os Imaginators oferecendo uma série de benefícios, incluindo desbloquear partes dos Imaginators, acesso a áreas especiais do jogo e a concessão de Técnicas Secretas para todas as Classes de Batalha.


História

No passado, os Anciões usavam uma magia chamada Mind Magic para criar as Skylands. Após os eventos do último jogo, houve uma era de paz. No entanto, um ser misterioso rouba um livro da Academia Skylanders e é revelado que ele trabalha para Kaos, que procura usar o poder da Mind Magic para criar um exército de Doomlanders e conquistar as Skylands. Para impedir isso o Mestre Eon chamou um Portal Master (o jogador) para que ele também possa usar Mind Magic para criar novos Skylanders, os Imaginators, para combater os Doomlanders, e para ajudar a treina-los o Eon enviou ao Portal Master os Senseis para puderem ajudá-los.


Roster

Este jogo tem 31 personagens jogaveis, todos Senseis, dos quais 11 deles são vilões dos jogos anteriores.

Assim como o jogo anterior este tambem tem 2 Skylanders crossovers, desta ves sendo com a Franquia Crash Bandicoot da própria Activision.Tendo 3 Skylanders para os elementos Fogo, Magia, Água, Ar, Tecnologia, Terra, e Zombie, 4 Skylanders para os elementos Tecnologia, e Vida, 2 Skylander para os elementos Luz, e Escuridão e 1 Skylanders do Kaos.
